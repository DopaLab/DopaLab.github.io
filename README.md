# PolyPericles Portfolio

A responsive, static portfolio for PolyPericles and DopaLAb. It is designed to deploy directly to GitHub Pages with no build step.

## Publish on GitHub Pages

1. Upload this folder to the root of your GitHub repository.
2. In the repository, open **Settings > Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select your main branch and the root folder, then save.

## Add another app

Open `script.js` and add an item to the `sideProjects` list at the top. Add a name and icon letter, and the site creates the card automatically. A URL, description, and accent color are optional.

## Main files

- `index.html`: portfolio content and card layout
- `styles.css`: visual design and responsive behavior
- `script.js`: contact flip interaction and customizable app cards
- `assets/`: portrait, transparent DopaLAb and Vybrik logos, Vybrik app screen, signature, and Misa artwork
